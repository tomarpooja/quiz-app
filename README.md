# Quiz App

A Django-based quiz app with a modern dark UI.

## Features
- Timer and progress bar
- Stylish dark theme
- Quiz result with PASS/FAIL
- Restart quiz option

## How to Run
```bash
git clone https://github.com/tomarpooja/quiz-app.git
cd quiz-app
python -m venv venv          # optional
venv\Scripts\activate        # Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
Open in browser: http://127.0.0.1:8000/
